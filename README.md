# jega
Full-stack fitness app. Built over summer 2026 as a personal project.

## Function
- Circuit-based workout tracking with timers and audio cues
- Auto-progression system that adjusts weights week over week based on performance
- AI coaching layer for form help, injury questions, and exercise alternatives
- Spotify integration for in-app music control
- Weekly program auto-generation via background jobs
- Analytics dashboard for tracking volume, PRs, and streaks

## Tech stack
- **Backend:** Node.js, Express, PostgreSQL, Redis, BullMQ
- **Frontend:** React, TypeScript
- **Auth:** JWT, bcrypt
- **Integrations:** Spotify API, Claude API
